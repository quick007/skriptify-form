Welcome to the Creators Program

The goal of this is to allow a select group of users to access the website early and help us populate it with avariety of skripts before it's official release. By participating in the Creator Program, you will have the opportunity to share your skripts with the community and gain recognition as one of the first contributors to the platform. In order to onboard users, we've created this simple website with an FAQ an an onboarding form. Please fill out this form, then look at the FAQ if you have any questions about this program.

FAQ

When will I be able to upload skripts?

You will be able to upload skripts once we finish key parts of the website. You will have access to the website in a beta phase, so some features may be broken. We ask that you report any bugs to the staff team.

Will I be able to sell skripts?

We currently have plans to allow users to sell skripts, but that feature will most likely not be present in the initial release.

Can I make money without charging users for skripts?

In the future, we might experiment with some sort of advertising, which you could make money from. This feature will not be present in the initial release.

How will I be able to log in?

All users will be able to log in with GitHub or Discord. As a creator, we highly recommend logging in with GitHub as it will allow you to upload skripts faster, take advantage of possible future features, and use versioning. You may also add your Discord username to your profile.

What methods can skripts be uploaded in?

If signed in to GitHub, users will also be able to connect a reposetory or repo. We highly recommend one of these options, as they allow for more flexability. You can also upload files, although you may not have access to all the features compared to a user using the GitHub connection.
